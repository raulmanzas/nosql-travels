# DynamoDB

Since DynamoDB is fully managed by AWS, there are not many administration tasks to practice locally. The code in this repository lets you run the official DynamoDB container on your machine so you can learn data modelling and play interact with the database just as you would if you were running on AWS.

Learn the concepts and come back to practice!
* https://docs.aws.amazon.com/amazondynamodb/latest/developerguide/HowItWorks.CoreComponents.html
* https://docs.aws.amazon.com/amazondynamodb/latest/developerguide/ql-reference.html
* https://docs.aws.amazon.com/amazondynamodb/latest/developerguide/workbench.html
